## 📌 주요 기능
- 최신 인기 음악과 개인화된 추천 음악 제공
- 곡에 따라 바뀌는 예쁜 음악 플레이어
- 최신 뮤직비디오를 둘러보며 좋아하는 노래를 바로 플레이리스트에 저장
- 편리한 플로팅 미니 뮤직 플레이어 사용
- 다양한 애니메이션과 햅틱 반응
